Generate Hibernate Mapping File

1. Click "Hibernate Code Generation Configurations" in the hibernate perspective.

2.In the Main tab page, select your console configuration which is hibernate(1) here. Set the ouput directory and check the "Reverse engineer from JDBC Connection"

OutputDirectory: the directory of package i.e. \crawler\test
Package: name of package i.e. com.bing.model

3.In the Exporters tab page, select to generate domain code, mapping files. If you want to use annotations, please check the "Generate EJB3 annotations".

4. XML mapping files
