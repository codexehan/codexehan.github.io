Install Hibernate JBoss Tool

1. Open eclipse and click Help -> Install New Software...

The update site of JBoss is http://download.jboss.org/jbosstools/oxygen/stable/updates/ Oxygen is the version of my Eclipse.

2.Type "hibernate" in the text area and select all. Click finish to install step by step.
