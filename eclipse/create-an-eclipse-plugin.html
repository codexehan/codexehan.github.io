Create an eclipse plugin

1. Use the Eclipse installer to install the Eclipse for RCP and RAP Developers.

2. Create a new plugin project named "CustomPlugin" and select the Hello World template

3. Open the plugin.xml, select the extension tab and delete all the existing extensions

4. Delete the customplugin.handlers package from the project explorer. Till now, an empty eclipse plugin has been created.
