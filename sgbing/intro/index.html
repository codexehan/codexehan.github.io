Since visiting SGBING running on my localhost depends on unstable reverse proxy software, it may not be accessed sometimes. You can briefly know some functions from the captures below.

SGBING is a website to provide service for Chinese in Singapore. Currently, the implementation of introductions of NTU and NUS has been complemented. But the data of majors has not been loaded to the database. Thus, data shown on the website is example data now.

There are some details cannot be explained from captures, such as password reset, sign up, account activation (the account can only exist in the database for 24 hours if it is not activated by email), crawling offer from Gter bbs, and so on.

In future, I will add more functions such as recommendation system (recommend friend to the user according to his following or basic information), the chat system and other review system.
